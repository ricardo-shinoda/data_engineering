## Install venv

```bash
sudo apt install python3-venv
```

Criar o ambiente virtual

```bash
python3 -m venv nome_do_ambiente
```

Nome do ambiente pode ser:
```bash
python3 -m venv venv
```

Ativar o ambiente:
```bash
source venv/bin/activate
```

Desativar
```bash
deactivate
```
